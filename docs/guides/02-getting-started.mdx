# Getting Started with PE Camera Rigs

Welcome to **PE Camera Rigs**! This addon provides four professional camera systems for Blender 4.0+. This guide gives you a quick overview of each rig and helps you choose the right one for your project.

**Already installed?** Great! Press `N` in the 3D Viewport and look for the **"PE Cams"** tab.

**Need to install?** See the [Installation Guide](./installation.mdx).

---

## Quick Overview

PE Camera Rigs includes **four specialized camera systems**, divided into two categories:

### Interactive Rigs (Real-Time, Modifier-Driven)

Perfect for quick setup and live feedback. Orbit controls in Modifier Properties, Isometric in Object Properties.

| Rig | Best For | Setup Time |
|-----|----------|------------|
| **[Orbit Camera](./orbit-camera.mdx)** | Product turntables, character showcases | 30 seconds |
| **[Isometric Camera](./isometric-camera.mdx)** | Game art, technical illustrations, architecture ‚ö° Real-time updates! | 30 seconds |

**How they work:** Click a button ‚Üí Controller object appears ‚Üí Adjust settings ‚Üí Done!

### Workflow Rigs (4-Step Production Pipeline)

Professional VR content creation with crash-safe rendering.

| Rig | Best For | Setup Time |
|-----|----------|------------|
| **[VR180](./vr180-workflow.mdx)** | Stereoscopic 180¬∞ VR for YouTube/Meta Quest | 5-10 minutes |
| **[VR360](./vr360-workflow.mdx)** | Monoscopic 360¬∞ panoramic VR | 5-10 minutes |

**How they work:** 4 numbered buttons guide you through: Scene Setup ‚Üí Render ‚Üí Composite ‚Üí Export

## Quick Start Tutorials

### Orbit Camera (2 minutes)

**Goal:** Create a spinning turntable animation

1. In **PE Cams** sidebar, find **"Orbit Camera"** panel
2. Select preset: **Product Turntable**
3. Click **"Add Orbit Rig"**
4. Select `Orbit_Controller` ‚Üí Go to **Modifier Properties** tab
5. Set **Orbit Radius** to your liking (distance from center)
6. Press **Spacebar** to preview animation
7. Press **Numpad 0** to look through camera

**Done!** The camera automatically rotates 360¬∞ over your animation duration.

**Next:** [Full Orbit Camera Guide](./orbit-camera.mdx)

---

### Isometric Camera (2 minutes)

**Goal:** Create a true isometric view

1. In **PE Cams** sidebar, find **"Isometric Camera"** panel
2. Select preset: **True Isometric**
3. Click **"Add Isometric Rig"**
4. Select `Isometric_Controller` ‚Üí Go to **Object Properties** tab
5. Find **"Isometric Camera Settings"** panel
6. Adjust **Ortho Scale** to zoom in/out (updates in real-time!)
7. Move the controller object to frame your scene
8. Press **Numpad 0** to look through camera

**Done!** You have a perfect isometric projection camera with interactive controls.

**Next:** [Full Isometric Camera Guide](./isometric-camera.mdx)

## Understanding the Two Rig Types

### Interactive Rigs (Orbit & Isometric)

**Technology:** Geometry Nodes
**Control Method:** Modifier properties
**Animation:** Automatic (driven by timeline)
**Best For:** Fast iteration, real-time feedback

**Key Features:**
- All settings in one place (Modifier Properties tab)
- Changes update instantly in viewport
- Non-destructive - modify anytime
- Can be keyframed for complex animations
- No multi-step workflow needed

**Example Use:**
```
Add Rig ‚Üí Adjust Modifier ‚Üí Render
```

---

### Workflow Rigs (VR180 & VR360)

**Technology:** Python operators + Compositor
**Control Method:** Numbered button workflow
**Animation:** Manual timeline control
**Best For:** Production pipelines, crash-safe rendering

**Key Features:**
- 4-step guided workflow
- Crash-safe EXR rendering (resume if interrupted)
- Automatic compositor setup
- Separate render/composite/export steps
- Production-tested for large projects

**Example Use:**
```
Step 1: Create Scene
Step 2: Render Sequences (can pause/resume)
Step 3: Setup Compositor (tweak color/effects)
Step 4: Render Final Video
```

## Common Questions

### Can I use multiple rigs in one scene?
**Yes!** All rigs can coexist. You can even combine them (e.g., orbit around an isometric view).

### Do I need to know Geometry Nodes?
**No.** The Orbit and Isometric rigs create the nodes for you. Just adjust the modifier sliders.

### Can I edit the camera after creating it?
**Yes.** Interactive rigs: adjust modifier properties anytime. VR rigs: re-run workflow steps as needed.

### Where are my render outputs saved?
**Default:** Relative to your .blend file (`//pe_renders/`). Change in addon preferences or workflow settings.

### Can I keyframe the camera settings?
**Yes** for Interactive Rigs - hover over any modifier value and press `I` to keyframe.
**Manual** for VR Rigs - animate objects/lights normally, cameras are fixed to workflow.

## Tips for Success

**For Interactive Rigs:**
- Look through the camera (`Numpad 0`) before rendering
- Select the controller object to access modifier settings
- Use presets as starting points, then customize
- Keyframe modifier values for complex animations

**For VR Workflow Rigs:**
- Complete all 4 steps in order - don't skip!
- Step 2 can take hours - plan accordingly
- Test with low samples/resolution first
- Keep .blend file saved (relative paths use its location)

**General:**
- Press `N` if you don't see the sidebar
- Check System Console (`Window` ‚Üí `Toggle System Console`) for errors
- Save often (especially before long renders)
