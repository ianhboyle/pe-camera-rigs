## [1.0.0] - 2025-12-09

### Added

- **Orbit Camera** - Procedural turntable animations with geometry nodes
  - 6 easing presets (Linear, Ease In/Out, Sine, etc.)
  - Customizable orbit radius, height, rotation
  - Real-time viewport preview

- **Isometric Camera** - True isometric and axonometric projections
  - 7 projection presets (Game 2:1, True Isometric, Dimetric, etc.)
  - Custom angle support
  - Orthographic scale control

- **VR180 Workflow** - Professional stereoscopic 180° VR production
  - 4-step crash-safe workflow
  - Side-by-side stereo rendering
  - IPD (interpupillary distance) control
  - EXR sequence rendering
  - Automated compositor setup
  - YouTube-ready MP4 export

- **VR360 Mono Workflow** - Monoscopic 360° panoramic VR
  - 4-step crash-safe workflow
  - Equirectangular projection
  - EXR sequence rendering
  - Automated compositor setup
  - YouTube-ready MP4 export

- **Unified UI** - Single "PE Cams" panel in 3D Viewport sidebar
- **Resolution presets** - YouTube 4K, 5K, 8K; Meta Quest 2/3/Pro
- **Scene setup helpers** - Lighting presets, cyclorama, reference objects
- **GPU detection** - Automatic GPU rendering enablement
